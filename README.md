# Codespaces_Extensiones
Extensiones listas para codespace según lenguaje.

Cada vez que se abre un repositorio nuevo, Codespaces pregunta si se quiere instalar las extensiones recomendadas. En vez de eso se puede crear un json personalizado con las extensiones deseadas según lenguaje. 
Pasos a seguir:

# 1- Crear un archivo .json
Crea .devcontainer/devcontainer.json en el repositorio deseado.
Dentro, añade esto para que siempre que se abra ese proyecto, las extensiones ya estén instaladas.

Adjunto varios .json según lenguajes (solo hay que cambiar el nombre o el contenido si se desea)
