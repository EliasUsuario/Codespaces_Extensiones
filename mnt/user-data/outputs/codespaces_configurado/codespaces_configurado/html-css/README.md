# 🎨 HTML & CSS — Codespace listo para usar

## Cómo empezar

1. Abre Codespaces desde **esta carpeta**
2. Espera a que el entorno se prepare (menos de 1 minuto)
3. Haz clic derecho sobre `index.html` → **"Open with Live Server"**

Los cambios se actualizan automáticamente cada vez que guardas. 🚀

---

## ¿Qué viene incluido?

| Extensión | Para qué sirve |
|---|---|
| Live Server / Live Preview | Recarga automática en el navegador |
| HTML CSS Support | Autocompletado de clases CSS en HTML |
| Auto Rename Tag | Renombra la etiqueta de cierre automáticamente |
| Auto Close Tag | Cierra etiquetas HTML solas |
| CSS Peek | Salta a la definición CSS al hacer clic |
| IntelliSense for CSS classes | Sugiere clases CSS mientras escribes |
| Tailwind CSS IntelliSense | Soporte para Tailwind (si lo usas) |
| Path IntelliSense | Autocompletado de rutas de archivos |
