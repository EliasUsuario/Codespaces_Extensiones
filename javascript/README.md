# 🟨 JavaScript en Codespaces

Este entorno configura automáticamente Node.js 20 con todo lo necesario.

## ¿Qué incluye?
- Node.js 20
- Extensiones: ESLint, Prettier, JavaScript Snippets
- Formato automático al guardar

## ¿Cómo usarlo?
1. Abre esta carpeta en GitHub Codespaces
2. Espera a que el contenedor se construya
3. Ejecuta el archivo de ejemplo:
```bash
   node index.js
```
