# 🟨 JavaScript — Codespace listo para usar

## Cómo empezar

1. Abre Codespaces desde **esta carpeta**
2. Espera a que el entorno se prepare (1-2 minutos la primera vez)
3. Ejecuta el archivo de ejemplo:

```bash
node index.js
```

¡Listo! Ya tienes Node.js y JavaScript funcionando.

---

## ¿Qué viene incluido?

| Extensión | Para qué sirve |
|---|---|
| ESLint | Detecta errores y malas prácticas |
| Prettier | Formateo automático al guardar |
| npm IntelliSense | Autocompletado para imports de npm |
| IntelliCode | Sugerencias con IA |
| Auto Rename/Close Tag | Facilita el trabajo con HTML en JS |
| Quokka.js | Ejecuta JS en tiempo real mientras escribes |
| DotENV | Resaltado de archivos `.env` |

**Versión de Node.js:** 20 LTS  
**npm:** incluido

---

## Instalar dependencias

```bash
npm install nombre-del-paquete
```

---

## Crear un proyecto Node.js desde cero

```bash
npm init -y
```

---

## Estructura sugerida

```
mi-proyecto/
  .devcontainer/
    devcontainer.json   ← copia esto a tu repo
  index.js
  package.json          ← opcional
  .env                  ← opcional, para variables de entorno
  README.md
```
