# ☕ Java en Codespaces

Este entorno configura automáticamente Java 21 con todo lo necesario.

## ¿Qué incluye?
- Java 21
- Extensiones: Java Extension Pack, Maven
- Formato automático al guardar

## ¿Cómo usarlo?
1. Abre esta carpeta en GitHub Codespaces
2. Espera a que el contenedor se construya
3. Compila y ejecuta el archivo de ejemplo:
```bash
   javac Main.java
   java Main
```
