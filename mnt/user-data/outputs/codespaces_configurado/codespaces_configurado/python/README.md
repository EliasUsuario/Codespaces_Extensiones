# 🐍 Python — Codespace listo para usar

## Cómo empezar

1. Abre Codespaces desde **esta carpeta**
2. Espera a que el entorno se prepare (1-2 minutos la primera vez)
3. Ejecuta el archivo de ejemplo:

```bash
python main.py
```

¡Listo! Ya tienes Python funcionando.

---

## ¿Qué viene incluido?

| Extensión | Para qué sirve |
|---|---|
| Python + Pylance | Autocompletado inteligente y análisis de tipos |
| Black Formatter | Formateo automático al guardar |
| Jupyter | Soporte para notebooks `.ipynb` |
| autoDocstring | Genera docstrings automáticamente |
| IntelliCode | Sugerencias de código con IA |

**Versión de Python:** 3.12

---

## Instalar dependencias

```bash
pip install nombre-del-paquete
```

Para proyectos con `requirements.txt`:

```bash
pip install -r requirements.txt
```

