# 🎨 HTML & CSS en Codespaces

Este entorno configura automáticamente un servidor local con Live Preview.

## ¿Qué incluye?
- Live Preview (servidor local en el navegador)
- HTML Snippets y CSS IntelliSense
- Script JS básico de complemento
- Formato automático al guardar

## ¿Cómo usarlo?
1. Abre esta carpeta en GitHub Codespaces
2. Espera a que el contenedor se construya
3. Abre `index.html` y haz clic en **Go Live** en la barra inferior

## Estructura
```
/html-css
├── .devcontainer/
│     └── devcontainer.json
├── index.html
├── styles.css
├── script.js
└── README.md
```
