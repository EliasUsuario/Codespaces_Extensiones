# 🐍 Python en Codespaces

Este entorno configura automáticamente Python 3.12 con todo lo necesario.

## ¿Qué incluye?
- Python 3.12
- Extensiones: Pylance, Black Formatter, Jupyter
- Formato automático al guardar

## ¿Cómo usarlo?
1. Abre esta carpeta en GitHub Codespaces
2. Espera a que el contenedor se construya
3. Ejecuta el archivo de ejemplo:
```bash
   python main.py
```
